## ❓ Frequently Asked Questions

Read these answers if you run into common issues during your setup.

**Does this software work on Mac?**
Currently, RemotePC-2026 only supports Windows 10 and Windows 11.

**Can I run multiple remote sessions at once?**
Yes, the application allows you to connect to multiple computers if you have them linked to your account.

**Is my data visible to others?**
Your connection uses end-to-end encryption. Only you hold the keys to access your remote computer.

**What do I do if the connection is slow?**
Close other applications on your computer that use high amounts of data, such as streaming services or large file downloads. A stable internet connection helps improve performance.

**Can I transfer files while connected?**
Yes, use the file transfer button in the toolbar of the remote window to move files between your local and remote computer.

## 🔧 Troubleshooting

If you encounter problems, try these manual checks first.

*   Check if your internet connection works on both computers.
*   Verify that the RemotePC-2026 application is open and logged in on the remote machine.
*   Restart both computers if the connection fails to establish.
*   Ensure that no firewall or security software blocks the application. You might need to add RemotePC-2026 to your allowed programs list in your Windows Defender settings.
*   Check for updates by clicking the Help menu in the application. Always use the latest version to ensure compatibility.

## 📋 Best Practices

Use these tips to get the most out of your remote experience.

1. Set your remote computer to never sleep. You cannot connect to a computer that is powered off or in sleep mode. 
2. Change your power settings in Windows to High Performance to ensure the computer stays active.
3. Use a wired Ethernet connection instead of Wi-Fi for your host computer to increase stability.
4. If you have multiple monitors, you can toggle between them using the view options located in the remote toolbar.
5. Minimize the remote window to restore access to your local files at any time.
